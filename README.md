## Запуск проекта

### Соберите и запустите контейнеры

Запустите команду для сборки и запуска приложения:

```bash
docker compose up --build
```

### Доступ к приложению

После успешного запуска приложение будет доступно по адресу:

```
http://localhost:8080
```
